## MedPredict Frontend (React + Vite)

### Requirements
- Node.js 18+ installed

### Setup
```bash
cd frontend
npm install
```

### Configure API
Create `frontend/.env` (or copy from `.env.example`) and set:
```
VITE_API_BASE_URL=http://localhost:8000
```

### Run
```bash
npm run dev
```

Open `http://localhost:5173`
